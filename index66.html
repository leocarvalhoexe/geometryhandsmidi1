Configurações OSC

Configure o endereço IP e a Porta do servidor WebSocket OSC. Estas configurações são salvas localmente no seu navegador e aplicadas ao clicar em "Salvar e Reconectar".

Ao abrir via file:///, o IP padrão é 127.0.0.1. Para uso em rede (ex: celular), insira o IP do computador onde osc_relayXX.py está rodando.

Certifique-se que o script Python está escutando na porta configurada (padrão 8080) e em 0.0.0.0.

Manual do Usuário - MIDI Shape Manipulator v66

Visão Geral

O MIDI Shape Manipulator (MSM) é uma aplicação web interativa que transforma gestos de mão, capturados pela webcam, em música e visualizações dinâmicas. Controle formas na tela, gere notas MIDI, mensagens OSC, e som diretamente no navegador, com gravação de áudio interna e um HUD de gravação aprimorado.

Ideal para performances audiovisuais, exploração sonora e aprendizado sobre interação gestual.

Novidades da v66

  • Interação por Gestos Aprimorada: Detecção de gestos otimizada para maior precisão e suavidade na manipulação das formas. Feedback visual opcional para gestos reconhecidos.
  • Layout de Controles Reorganizado: Botões e sliders agrupados por categorias (Áudio, MIDI, Geometria, OSC) com uso de ícones e cores para facilitar a identificação. Botões de acionamento mais interativos.
  • Interface Mais Intuitiva: Adição de tooltips explicativos, indicadores de status e feedback visual imediato para ações do usuário. Melhorias na responsividade para dispositivos móveis.

Novidades da v63 (Anterior)

  • Mapeamento Personalizado de Gestos para Parâmetros: Nova interface no painel do sintetizador para mapear deformações da forma (como Liquify, Raio, Lados) a parâmetros do sintetizador (Filtro, LFO, Efeitos). Configurações salvas localmente.
  • Variações Aleatórias no Arpejador: Adicionados controles de Randomness, Swing e Ghost Note Chance (%) ao painel de configurações do arpejo, permitindo maior variação rítmica e melódica.

Iniciando

  1. Permissão da Câmera: Ao abrir a página, seu navegador solicitará permissão para usar a webcam. Conceda para ativar a detecção de mãos.
  2. Áudio Interno:
    • Por padrão, o áudio interno está habilitado. Você deverá ouvir som assim que os gestos começarem a gerar notas.
    • Clique em qualquer lugar da página ou pressione uma tecla para garantir que o contexto de áudio do navegador seja ativado (política de autoplay).
    • Use o botão "🔊 Áudio Interno" (Shift+V) para ligar/desligar.
    • Ajuste o volume, forma de onda, ADSR e outros parâmetros do sintetizador no painel à direita ou no modal de "⚙️ Configurações".
  3. Saída MIDI (Opcional):
    • Para enviar notas MIDI para um DAW (Digital Audio Workstation) ou sintetizador de hardware, vá em "⚙️ Configurações" e selecione sua porta de saída MIDI desejada.
    • Certifique-se que o botão "🎹 MIDI" na barra lateral está ativo (ON). Atalho: M.
  4. Conexão OSC (Opcional):
    • Para enviar mensagens OSC (Open Sound Control), configure o IP e a Porta do servidor OSC no modal "📡 OSC Config". O script osc_relayXX.py (incluído no repositório, execute com Python) pode ser usado como um servidor de relay local.
    • Após configurar, clique em "Salvar e Reconectar". O status da conexão aparecerá no HUD.

Interface Principal

  • Barra Lateral (Esquerda): Acesse todas as funcionalidades principais. Clique no ícone ☰ para abrir/fechar.
  • Painel de Controle do Sintetizador (Direita): Controles manuais para o sintetizador interno. Use o botão 🎛️ no canto superior direito para mostrar/ocultar.
  • Painel de Controle do Arpejador (Direita): Controles para o arpejador. Use o botão 🎼 no canto superior direito para mostrar/ocultar.
  • Canvas Principal: Área onde as formas são desenhadas e interagem com seus gestos.
  • HUD (Heads-Up Display): Painel no canto inferior direito com informações de status em tempo real (MIDI, OSC, Áudio, gestos, etc.). Use o botão "ℹ️ Info HUD" na barra lateral para mostrar/ocultar.

Controles e Funcionalidades (Botões da Barra Lateral Esquerda)

Os botões estão agrupados por funcionalidade para facilitar o acesso.

Geral & Visualização
  • ℹ️ Manual: Abre esta janela de informações (Shift+I).
  • ℹ️ Info HUD: Mostra/oculta o painel de informações (HUD).
  • 🌙 Tema: Alterna entre tema claro e escuro (Shift+T).
  • 👓 Espectador: Desativa o controle por gestos e para a geração de notas/OSC (Shift+S).
Áudio & MIDI
  • 🔊 Áudio ON/OFF: Ativa/Desativa o sintetizador de áudio interno (Shift+V).
  • 🎹 MIDI ON/OFF: Ativa/Desativa o envio de mensagens MIDI (M).
  • 🎤 MIDI In ON/OFF: Ativa/Desativa o feedback de mensagens MIDI recebidas para o log OSC (Shift+F).
OSC & DMX
  • 📡 OSC Config: Configura IP e Porta para comunicação OSC (Shift+K).
  • 🎶 Sync DMX ON/OFF: Sincroniza notas MIDI como mensagens OSC formatadas para controle DMX (Shift+D).
  • ⏺️ Gravar OSC: Grava uma sequência de mensagens OSC enviadas pela aplicação (Shift+R).
  • ▶️ Loop OSC: Reproduz a sequência OSC gravada. Fica habilitado após uma gravação (Shift+P).
Ferramentas & Desenvolvimento
  • ⚙️ Configurações: Abre o modal para configurar Saída/Entrada MIDI, Câmera e parâmetros detalhados de Áudio Interno (Shift+C).
  • 💾 Presets: Gerencia presets de configuração das formas (Salvar, Carregar, Deletar, Importar/Exportar JSON) (Shift+B).
  • 🤖 Simulador ON/OFF: Ativa/Desativa um modo de simulação de gestos.

No painel do sintetizador (à direita), você encontrará os controles para gravação de áudio: "Gravar Áudio", "Pausar/Retomar" e "Salvar Áudio". O painel do arpejador (🎼) contém controles de BPM, estilo, etc.

Gestos e Interação com as Formas

O sistema detecta até duas mãos. Cada mão (ou par de mãos, no modo "one_person") pode controlar uma forma geométrica.

  • Posicionamento da Forma: O centro da forma segue a posição média dos pulsos.
  • Controle de Raio (Tamanho):
    • Modo "one_person": Junte polegares e afaste/aproxime as mãos com indicadores curvados.
  • Controle de Lados (Complexidade da Forma):
    • Modo "one_person": Mão esquerda, pinça polegar-indicador. Distância controla lados (3-20, ou 100 para círculo).
    • Quando a forma tem 100 lados (círculo), as notas geradas percorrem todo o espectro MIDI (0-127) de forma cromática, ignorando a escala selecionada.
  • Distorção / "Liquify" / Efeitos:
    • Modo "one_person": Mão direita, proximidade dos dedos aos vértices causa distorção e modula CCs MIDI.
  • Geração de Notas: Baseada no "Modo de Nota" (Sequencial, Arpejo, Acorde, Random Walk).
    • Se a forma for um círculo (100 lados), os modos Sequencial e Arpejo (e a nota base do Acorde) mapearão a progressão ao longo da circunferência para o range MIDI completo (0-127).

Solução de Problemas Comuns

  • Sem som (Áudio Interno):
    • Verifique "🔊 Áudio" (ON), clique na página, aumente volume, verifique saídas do PC.
    • Certifique-se de que uma forma de onda válida (ex: Sine, Square, Noise) está selecionada.
  • Câmera não funciona:
    • Verifique permissões, outras apps usando a câmera, selecione câmera específica em "⚙️ Configurações".
  • OSC não conecta:
    • Verifique IP/Porta, script Python relay, firewall.

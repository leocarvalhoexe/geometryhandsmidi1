Configurações de Arpejo

Configurações de arpejo são aplicadas em tempo real.

Manual do Usuário - v33

Novidades da v33

  • Suporte a Mensagens OSC Binárias: O servidor Python (`osc_relay33.py`) agora decodifica e retransmite mensagens OSC binárias recebidas via WebSocket, além das mensagens JSON. Isso permite maior compatibilidade com softwares que enviam OSC em formato binário nativo.
  • HUD Funcional: A Heads-Up Display (HUD) na parte inferior da tela agora exibe o status "Sistema iniciado..." e outras mensagens relevantes do sistema.
  • Painel OSC e Config Operacionais: Os modais de Configurações (para MIDI) e Painel OSC (para envio de testes e logs) estão completamente funcionais e integrados.
  • Inicialização Automática: O sistema agora é carregado automaticamente quando a página é aberta (`window.onload`), inicializando MIDI, OSC, eventos e câmera.
  • Importação Correta do osc-js: A biblioteca `osc-js` está corretamente importada, garantindo a funcionalidade de comunicação OSC no lado do cliente.

Visão Geral - Novidades v32

Esta versão (v32) inclui correções cruciais de inicialização e melhorias de usabilidade e persistência de dados sobre a v31:

  • Botão "Sync DMX Notes": Espelha notas MIDI (internas ou externas via feedback) como mensagens OSC /dmx/note <nota> <velocidade>.
  • Gravador/Reprodutor de Loop OSC: Botões para gravar uma sequência de mensagens OSC enviadas e reproduzi-las em loop com duração configurável.
  • Recebimento de BPM Externo: Handler OSC /global/setExternalBPM <bpm> para sincronizar o tempo do arpejo com softwares externos.
  • Feedback MIDI para OSC: Encaminha notas e CCs MIDI recebidos de dispositivos externos como mensagens OSC (/midi/in/noteOn, /midi/in/cc, etc.).
  • Painel de Teste OSC: Interface para enviar mensagens OSC manualmente e visualizar logs.
  • Modo "Spectator": Visualiza as interações sem permitir controle manual, ideal para instalações.
  • Correção no Servidor Python: osc_relay30.py agora lida corretamente com mensagens binárias OSC enviadas pelo navegador via WebSocket, além de JSON.
  • Melhorias na Interface: Botões reorganizados, tooltips e estilo visual atualizado.

Mantém todas as funcionalidades da v29, como controle de formas, modos de nota, MIDI, OSC, etc.

Controles e Botões Adicionais (v30)

  • Sync DMX Notes: (Barra de Controles) Ativa/desativa o envio de mensagens /dmx/note.
  • Gravar Loop OSC / Tocar Loop OSC: (Barra de Controles) Controlam a gravação e reprodução de sequências OSC. A duração do loop pode ser definida no Painel OSC.
  • Modo Espectador: (Barra de Controles) Alterna o modo de visualização passiva.
  • MIDI In (Feedback): (Barra de Controles) Ativa/desativa o encaminhamento de MIDI In para OSC. A porta de entrada é selecionada em ⚙️ Config.
  • Painel OSC (Botão "📡 Painel OSC"): Abre um modal para:
    • Enviar mensagens OSC customizadas (endereço, argumentos).
    • Configurar a duração do loop OSC gravado.
    • Visualizar um log de mensagens OSC (enviadas pelo painel/loop, recebidas via UDP).

Comunicação OSC (Adições v30)

Novas mensagens OSC enviadas pelo sistema:

  • /dmx/note <nota_midi> <velocidade>: Enviada quando "Sync DMX Notes" está ativo.
  • /midi/in/noteOn <canal> <nota> <velocidade>
  • /midi/in/noteOff <canal> <nota>
  • /midi/in/cc <canal> <numero_cc> <valor_cc>
  • /midi/in/pitchbend <canal> <valor_bend_14bit>
  • /global/state/dmxSyncMode <0|1>
  • /global/state/midiFeedbackEnabled <0|1>

Novas mensagens OSC recebidas pelo sistema:

  • /global/setExternalBPM <bpm_float_ou_int>: Define o BPM do arpejo. Enviar 0 ou nulo para desativar.

O script osc_relay30.py foi atualizado para lidar com mensagens binárias OSC e JSON do navegador.

Funcionalidades da v29 (Resumo Mantido)

Manipulação de formas, geração MIDI (escalas, modos Sequencial/Arpejo/Acorde, Staccato/Pulso), controle de efeitos (Pitch Bend, Reverb, Delay, Pan, Brilho, ModWheel, Resonance), modo 1 ou 2 pessoas, janela de saída visual, configurações MIDI, e comunicação OSC básica com osc_relay.

Teclas de Atalho Principais:

  • M: MIDI On/Off
  • P: Ativar/Desativar Modo Pulso (Não confundir com Shift+P para Loop OSC)
  • L: Staccato/Legato
  • S: Ciclar Escalas (Não confundir com Shift+S para Modo Espectador)
  • N: Ciclar Modos de Nota
  • V: Puxar Vértices On/Off
  • C: Alternar Modo de Acorde (Não confundir com Shift+C para Config)
  • Shift + I: Abrir/Fechar Manual
  • Shift + C: Abrir/Fechar Configurações
  • Shift + A: Abrir/Fechar Painel Arpejo
  • Shift + O: Abrir/Fechar Painel OSC
  • Shift + D: Sync DMX On/Off
  • Shift + F: MIDI Feedback On/Off
  • Shift + R: Gravar/Parar Loop OSC
  • Shift + P: Tocar/Parar Loop OSC
  • Shift + S: Modo Espectador On/Off
  • Esc: Fecha o modal ativo ou remove foco de inputs.

Atalhos são desabilitados no modo espectador e quando um campo de texto está focado (exceto Esc).
